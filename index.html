Свойства Flex-контейнера

display: flex;


display: -webkit-flex

Назначение: запуск механизма Flexbox

Возможные значения: flex / inline-flex;

flex

inline-flex

flex-direction: row;


-webkit-flex-direction: row;

Назначение: направление укладки элементов по осям

Возможные значения: row / row-reverse / column / column-reverse;

row

column

row-reverse

column-reverse

flex-wrap: nowrap;


-webkit-flex-wrap: nowrap;

Назначение: расположение элементов в одну или несколько строк

Возможные значения: nowrap / wrap / wrap-reverse;

nowrap

wrap

wrap

flex-flow: row nowrap;


-webkit-flex-flow-nowrap: nowrap;

Назначение: сокращенная запись для свойств flex-direction и flex-wrap

Возможные значения: см.свойства 2 и 3

justify-content: flex-start;


-webkit-justify-content-flex-start: nowrap;

Назначение: Выравнивание flex элементов по главной оси текущей строки

Возможные значения: flex-start / flex-end / center / space-between / space-around

flex-start

flex-end

center

space-between

space-around

-webkit-align-items-stretch;

Назначение: Выравнивание flex элементов вдоль поперечной оси

Возможные значения: stretch/ flex-start / flex-end / center / baseline

stretch

center

baseline

Назначение: Выравнивание flex элементов вдоль поперечной оси

Свойства Flex-элемента

order: 0;


-webkit-order-0;

Назначение: Определение порядка постановки flex элементов

Возможные значения: (целое число)

order 0 order 1

order: -1 order: 0 rder:2

flex-grow: 0;


-webkit-flex-grow-0;

Назначение: Указывает фактор растягивания flex элементов

Возможные значения: (целое неотрицательное число)

flex-grow: 1

flex-grow: 3

flex-shrink: 1;


-webkit-flex-shrink-0;

Назначение: Указывает фактор сужения flex элементов

Возможные значения: 0

flex-shrink: 1

flex-shrink: 0

flex-basis: auto;


-webkit-flex-basis auto;

Назначение: Указывает исходныйглавный размер flex элементов

Возможные значения: длина элемента в пикселях (к примеру, 240рх)

flex-base: auto

flex-base: 240px

flex: 0 1 auto;


-webkit-flex 0 1 auto;

Назначение: Сокращенная запись для свойств flex-grow, flex-shrink и flex-basis

Возможные значения: свойства 2,3 и 4

align-self: auto;


-webkit-align-self:auto;

Назначение: Определяет выравнивание элемента по умолчанию

Возможные значения: auto / flex-start / flex-end / center/ baseline / stretch

align-self: flex-start

align-self: flex-end
